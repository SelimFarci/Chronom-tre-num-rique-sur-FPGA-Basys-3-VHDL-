# Basys3 Chronomètre

Chronomètre numérique conçu en **VHDL** pour la carte **Basys 3**, développé avec **Vivado**.

## Description

Ce projet implémente un chronomètre numérique sur FPGA utilisant la carte Basys 3.  
Il utilise les **afficheurs 7 segments** pour afficher les secondes et les minutes et les **boutons de la carte** pour démarrer, arrêter et réinitialiser le chronomètre.

## Fonctionnalités

- Affichage du temps en minutes et secondes sur les afficheurs 7 segments.
- Boutons pour :
  - Démarrer / arrêter le chronomètre
  - Réinitialiser le chronomètre
- Implémentation entièrement en VHDL.
- Conçu pour la carte **Basys 3** (FPGA Artix-7).
